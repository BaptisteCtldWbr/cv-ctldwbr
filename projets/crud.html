Pour un TP, en cours, nous avons développé un site permettant de gérer une base de données avec les adhérents d'un club. On retrouve leur nom, prénom, date de naissance et photo. On peut ajouter, rechercher, modifier ou supprimer un adhérent.

Voici la liste non-exhaustive des fonctionnalités :

  • CRUD (Create, Read, Update, Delete)
  • Connexion, déconnexion avec les sessions PHP
  • Recherche d'un adhérent avec le nom, le prénom et/ou la date de naissance
  • Pagination
  • Upload de fichiers et vérification (photos)

Vous pouvez essayer par vous même, l'identifiant et le mot de passe est pré-saisi.
